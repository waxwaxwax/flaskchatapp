About This App

このアプリは Azure OpenAI を利用したチャット機能を備えています。ユーザーが自由にスレッドを作成し、対話形式でやり取りが可能です。

主な機能：

  • スレッド管理（複数の会話を個別に保持）
  • Azure OpenAI を使った自然な文章生成
  • Tailwind CSS を利用したレスポンシブUI


今後は以下のアップデートを予定しています：

  • ファイルアップロードと要約機能
  • 日本語以外の多言語対応
  • ユーザープロフィールのカスタマイズ


開発者： このアプリは個人開発で、Flask と Tailwind CSS、Azure OpenAI を活用しています。 何か不具合や改善点があれば、お気軽にご連絡ください。
